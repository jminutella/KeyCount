Key Count & Palindrome Program

Key Count

This text area is a key count. Each line in the textarea should be expected to contain a key, separated from a count by a colon character. The key may be any string not containing a colon. The count should be expected to be able to be parsed to an integer.

If a key is a palindrome, the result will be displayed in a dark blue color.

Please enter data in this format:

John: 3
Jane: 1
John: 2
Jane: 5

Results

Press the Calculate button to add up the key count totals.
